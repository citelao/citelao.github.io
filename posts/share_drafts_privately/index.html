How do I share drafts privately on the Internet?

Ideas:

  • Public — no one reads my blog anyway
  • Obfuscated (public) — use pages.dev domain + noindex.
  • Obfuscated (hidden) — use pages.dev + noindex + long hashes a la Google Photos to make it even harder to guess.
  • Encrypted
  • Password-protected site
  • Send PDFs
  • Send Word docs (via Pandoc)
  • Send Markdown
  Description Pros Cons
Public No one reads my blog anyway, just post publicly.
Obfuscated Publish to a "beta" domain that is not indexed by Google.
Cryptographically obfuscated Like obfuscated, but include a long random string in the URL, to make URL sniffing implausible (like Google Photos does).
Encrypted Like cryptographically obfuscated, but also encrypt the content. Include a password field & use Javascript to decrypt the page at runtime.
Custom password-protected site Don't even serve an encrypted version of the content: host a server & require users to login first.
Password-protected S3 (or equivalent) Have AWS or Azure manage the login for you.
Password-protected Cloudflare Pages Have Cloudflare manage the login for you.
Just send PDFs Don't bother with websites: use PDFs.
Just send DOCs Don't bother with websites: use Microsoft Word documents.
Just send Markdown Don't bother with websites: use markdown files.
